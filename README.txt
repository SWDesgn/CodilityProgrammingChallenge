An alphabet consists of letters "a", "b", "c".
A string is considered diverse if there are no 3 same consecutive letters.

The task is to write a function that takes 3 integers as input: A, B, and C and returns a string.

It should return any longest possible diverse string, with at most A letters "a", B letters "b", and C letters "c".

Example output for A=6, B=2, C=1

AABAABAAC

No unit testing was required.
